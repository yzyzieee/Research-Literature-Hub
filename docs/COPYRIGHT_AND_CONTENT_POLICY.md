# Copyright and Content Policy

Research Literature Hub manages metadata and team-authored literature context. It is not
a paper-distribution service.

## Public repository rules

- Do not commit copyrighted paper PDFs, book chapters, scans, or publisher downloads.
- Do not commit confidential manuscripts, peer-review material, or restricted datasets.
- Storage links may be committed only when the repository operator understands that
  the URL becomes public and the linked file remains protected by its own sharing
  permissions.
- Prefer DOI and publisher landing-page links for public bibliographic references.
- Keep quotations short, necessary, attributed, and compatible with applicable law.
- Write summaries, strengths, limitations, and team notes in original language.

## External PDF storage

The included Google Drive integration is an optional private storage adapter. Repository
operators are responsible for access control and for ensuring that every stored file may
lawfully be shared with the intended team.

A public literature record may link to an external file only when publishing that link
does not bypass access controls or violate a license, agreement, or institutional policy.

## Third-party rights

The project license covers the software and original project documentation. It does not
license third-party papers, publisher content, datasets, trademarks, or other referenced
works.

## Takedown requests

Repository maintainers should promptly investigate credible requests concerning
copyrighted, confidential, personal, or unlawfully shared material. When appropriate,
remove public links or content first, then preserve only the minimum audit information
needed to document the action.
