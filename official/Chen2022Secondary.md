## Summary
This paper introduces a novel Secondary Path-Decoupled Active Noise Control (SPD-ANC) algorithm that leverages deep learning to address the performance degradation and filter divergence issues in conventional ANC systems caused by nonlinearities in the secondary path. The proposed method utilizes two time-domain Convolutional Recurrent Networks (CRNs) to model the nonlinear secondary path and its inverse. This allows for the calculation of a secondary path-decoupled (SPD) error signal, which is then used to optimize an adaptive filter. Simulation results demonstrate that the SPD-ANC algorithm significantly outperforms traditional ANC methods, including FxLMS and deep learning-based approaches, in terms of noise cancellation performance and convergence speed, particularly under various nonlinear secondary path conditions.

## Problem
Conventional Active Noise Control (ANC) systems are effective for canceling low-frequency noise. However, their performance degrades significantly in the presence of nonlinearities in the secondary path (e.g., loudspeaker saturation). These nonlinearities cause the anti-noise signal to deviate from its intended form, leading to incomplete cancellation of the residual error. This incomplete cancellation can, in turn, cause the adaptive filter's control signal to increase uncontrollably, potentially leading to filter divergence and system instability. Existing methods to mitigate this, such as modifying cost functions or search directions, often rely on empirical parameters and lack robustness. Other approaches that model the nonlinear secondary path can be sensitive to the accuracy of the model or require pre-selected nonlinear functions.

## Method
The proposed SPD-ANC algorithm decouples the nonlinear secondary path from the adaptive filter's update process by introducing a secondary path-decoupled (SPD) error signal. This SPD error signal, denoted as $\hat{e}(n)$, is defined as:
$$ \hat{e}(n) = y(n) + \overleftarrow{S}(d(n)) $$ 
where $y(n)$ is the control signal, $d(n)$ is the unwanted noise signal, and $\overleftarrow{S}(\cdot)$ represents the reverse secondary path, which is the inverse model of the actual secondary path $S$. The goal is to minimize this $\hat{e}(n)$ instead of the conventional residual error $e(n) = d(n) + S[y(n)]$. Minimizing $\hat{e}(n)$ effectively removes the influence of the nonlinear secondary path $S$ from the error minimization process, leading to a more stable adaptive filter update.

To implement this, two deep neural networks (DNNs) are employed:
1.  **DNN-Based Secondary Path Module ($\varphi_S$):** This network models the forward nonlinear secondary path $S$. It takes the control signal $y(n)$ as input and estimates the anti-noise signal $y_s(n)$. The loss function for training this module is based on the Mean Squared Error (MSE) between the estimated and actual anti-noise signals, normalized by the power of the actual anti-noise signal:
    $$ L_s = 10 \log_{10} \frac{\sum_{n} |\hat{y}_s(n) - y_s(n)|^2}{\sum_{n} y_s^2(n)} $$ 
    where $\hat{y}_s(n)$ is the output of $\varphi_S$.
2.  **DNN-Based Reverse Secondary Path Module ($\varphi_{\overleftarrow{S}}$):** This network models the inverse of the secondary path. It takes the unwanted noise signal $d(n)$ as input and aims to produce an output that, when passed through the forward secondary path model $\varphi_S$, reconstructs the original unwanted noise signal. The constraint is $d(n - \beta) = \varphi_S(\varphi_{\overleftarrow{S}}(d(n)))$, where $\beta$ is a delay to ensure causality. The loss function for training this module is:
    $$ L_{\overleftarrow{S}} = 10 \log_{10} \frac{\sum_{n} |\hat{d}(n - \beta) - d(n)|^2}{\sum_{n} d^2(n)} $$ 
    where $\hat{d}(n - \beta)$ is the output of $\varphi_{\overleftarrow{S}}$ after passing through $\varphi_S$ and applying the delay.

Both DNNs are implemented using a time-domain Convolutional Recurrent Network (CRN) architecture. The forward secondary path model $\varphi_S$ is trained first, and its parameters are then frozen. The reverse secondary path model $\varphi_{\overleftarrow{S}}$ is trained using the outputs of the trained $\varphi_S$.

The adaptive filter is updated using the standard LMS algorithm, but with the SPD error signal $\hat{e}(n)$ and a potentially delayed reference signal $x(n-\beta)$:
$$ \omega(n+1) = \omega(n) - \mu \hat{e}(n-\beta) x(n-\beta) $$ 
where $\mu$ is the step size. The control signal is generated as $y(n) = \omega^T(n) x(n)$. The overall algorithm is detailed in Algorithm 1.

## Key results
The proposed SPD-ANC algorithm demonstrated superior performance compared to conventional methods like FxLMS, THF-NLFxLMS, and even a standard deep ANC approach. In time-invariant simulations, SPD-ANC achieved noise reduction levels comparable to non-causal deep ANC, while maintaining consistent performance across different primary and secondary path configurations and varying degrees of nonlinearity ($\eta^2$). Notably, it outperformed FxLMS, which degraded significantly in nonlinear conditions, and THF-NLFxLMS, which was sensitive to the accuracy of the pre-selected secondary path model. In time-varying simulations, SPD-ANC showed faster convergence and better performance when the primary path changed, unlike deep ANC which introduced additional noise during path transitions. The DNN-based secondary path modeling also outperformed traditional LMS and THF-NLFxLMS methods, achieving the best accuracy across all tested conditions due to DNNs' powerful modeling capabilities.

## Strengths
1.  **Robustness to Secondary Path Nonlinearities:** The core strength is its ability to effectively handle nonlinearities in the secondary path, which is a major limitation for traditional ANC systems. The SPD error signal decouples the adaptive filter update from these nonlinearities.
2.  **Improved Stability and Convergence:** By minimizing the SPD error, the algorithm achieves a more stable filter update process and faster convergence, even in challenging nonlinear environments.
3.  **Deep Learning Integration:** Leverages the powerful modeling capabilities of deep neural networks (CRNs) to accurately model both the forward and reverse secondary paths, surpassing traditional modeling techniques.
4.  **Adaptability:** The algorithm demonstrates good performance in time-varying conditions and across different primary paths, indicating adaptability.
5.  **Outperforms Baselines:** Consistently shows better noise cancellation performance and convergence speed compared to established ANC algorithms and a standard deep ANC approach.

## Limitations
1.  **Untrained Nonlinear Secondary Paths:** The paper notes that the performance is not guaranteed for nonlinear secondary paths that were not included in the training set. This suggests a potential limitation in generalization to completely novel nonlinearities.
2.  **Computational Complexity:** While not explicitly detailed as a limitation, the use of two deep neural networks likely increases the computational complexity compared to simpler adaptive filters like FxLMS, although it may be more manageable than some other deep learning approaches.
3.  **Training Data Dependency:** Like most deep learning methods, the performance relies on the quality and representativeness of the training data for modeling the secondary paths.

## Relevance to our group
This work is highly relevant to our group as it presents a state-of-the-art approach to active noise control that directly tackles a critical real-world challenge: secondary path nonlinearities. The integration of deep learning, specifically CRNs, for modeling complex acoustic paths and their inverses aligns with our interest in applying advanced machine learning techniques to audio signal processing problems. The proposed SPD-ANC algorithm offers a robust and effective solution that could be applied to various ANC applications, potentially improving performance in challenging acoustic environments. Understanding and implementing such advanced control strategies is crucial for developing next-generation audio systems.

## Notes
The paper uses time-domain Convolutional Recurrent Networks (CRNs) for modeling the secondary path and its reverse. The specific CRN architecture involves encoder/decoder blocks with Conv1d/Deconv1d layers and LSTM layers. The training uses Adam optimizer with a decaying learning rate. The secondary path modeling accuracy is evaluated using MSE, and noise cancellation performance is measured by Normalized Mean Square Error (NMSE) in dB. The reference signal is delayed by $\beta$ in the adaptive filter update equation (8) to align with the SPD error signal. Future work is suggested to assess robustness for untrained nonlinear secondary paths.
